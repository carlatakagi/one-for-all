## 🚀 Como executar o projeto
#### 🧭 Rodando a aplicação

```bash

# Clone este repositório em seu terminal
$ git clone git@github.com:carlatakagi/one-for-all.git

# Acesse a pasta do projeto no seu terminal/cmd
$ cd one-for-all

# Instale as dependências:
$ npm install

```
